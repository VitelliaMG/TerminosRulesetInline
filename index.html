Términos Ruleset e Inline

Términos Ruleset

Término Descripción Ejemplo
Selector El comienzo del conjunto de reglas usadas para apuntar al elemento que será estilizado. h {
color:black
}
Declaration Block El código entre corchetes (incluyéndolos) que contine la(s) declaración(es) CSS. h {
color: black

}
Declaration El nombre agrupado para una propiedad y par de valores que aplica estilo a un elemento seleccionado. h {
color: black

}
Property La primera parte de la declaración que muestra qué característica visual del elemento será modificada. h {
color: black

}
Value La segunda parte de la declaración que significa el valor de la propiedad. h {
color: black

}

Términos Inline Style

Término Descripción Ejemplo
Opening tag El comienzo de un elemento HTML. Se refiere al elemento que será estilizado. p style='color:blue';
Attribute Se usa para dar estilo en línea CSS a un elemento HTML. p style='color: blue';
Declaration Agrupa el nombre para una propuedad y su par de valores que aplica estilo a un elemento seleccionado. p style='color: blue;'
Property La primera parte de la declaración que nos muestra qué característica visual del elemento será modificada. p style='color: blue';
Value La segunda parte de la declaración que muestra el valor de la propiedad. p style='color: blue'
